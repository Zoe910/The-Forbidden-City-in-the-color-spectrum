# The Forbidden City in the color spectrum

## Introduction
Our project aims to create an interactive digital curation portal for the cultural relics of the Palace Museum. Users can explore the intersection of data, history, and the lively aesthetics of Chinese heritage through an immersive interface. We use color blocks as the core interactive tool to help users find and connect to related cultural relics.

At first, we planned to use images from the official Palace Museum website as our visual base. However, we found that the website restricts external network access during development. So we turned to Wikipedia instead. By deeply analyzing its public visual data, we decoded the traditional collection into a generative color system. This helps us translate and visualize cultural symbols in a digital way.
However, there are still some cultural relics pictures that cannot be found on Wikipedia.
